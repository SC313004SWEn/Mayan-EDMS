Create Docker Container
---
1. เข้าไปโฟลเดอร์ `docker/` คลิ๊กขวาเปิด Open in Integrated Terminal

2. ใน Terminal พิมพ์ `docker-compose up`

3. เช็คใน Docker Desktop
